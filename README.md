# 🎮 Tic Tac Toe AI

## 📌 Overview

This project is a Python-based implementation of the classic Tic Tac Toe game enhanced with Artificial Intelligence. The game features an interactive graphical user interface (GUI) built using Tkinter and includes multiple difficulty levels for the AI opponent.

The AI utilizes the Minimax algorithm to make optimal decisions, ensuring that the computer plays perfectly in hard mode.

---

## 🚀 Features

* 🖥️ User-friendly GUI using Tkinter
* 🤖 AI opponent with 3 difficulty levels:

  * Easy → Random moves
  * Medium → Rule-based (win/block logic)
  * Hard → Minimax algorithm (unbeatable AI)
* 🔄 Restart game functionality
* 📊 Score tracking system
* 🎯 Real-time game state updates

---

## 🧠 AI Implementation

The project uses the Minimax Algorithm, a recursive decision-making algorithm widely used in game theory.

* Evaluates all possible game states
* Maximizes AI winning chances
* Minimizes player advantage
* Guarantees optimal moves in hard mode
